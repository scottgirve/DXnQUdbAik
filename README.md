# 项目简介

## 核心功能
- 用户管理
- 音乐管理
- 评分管理
- 音乐搜索
- 音乐评价
- 音乐推荐接收

## 适用场景
- 音乐爱好者
- 音乐平台
- 音乐教育机构

## 技术特点
- 采用 Python 和 Django 进行开发，确保系统稳定性和扩展性
- 使用协同过滤算法进行音乐推荐，提高推荐准确性
- 集成 Mysql 数据库，保障数据存储和查询效率
- 基于 wordcloud 技术生成音乐词云，提供视觉化音乐分析
